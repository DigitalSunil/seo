# Schema Markup Guide

## What is Schema Markup?

Schema Markup is structured data that helps search engines understand the content of your website.

It improves eligibility for rich results and provides better context about your pages.

---

## Common Schema Types

### Organization

Describes your business.

### Person

Provides author information.

### LocalBusiness

Essential for local SEO.

### FAQ

Displays questions and answers.

### Article

Used for blog posts.

### Breadcrumb

Shows page hierarchy.

### Review

Displays ratings and reviews.

---

## Why Schema Matters

- Better search visibility
- Rich results
- Improved content understanding
- Stronger entity recognition
- Better AI search compatibility

---

## Best Practices

- Validate schema using Rich Results Test.
- Keep schema updated.
- Match structured data with visible page content.
- Use JSON-LD format whenever possible.

---

## Useful Tools

- Google Rich Results Test
- Schema Markup Validator
- Google Search Console
