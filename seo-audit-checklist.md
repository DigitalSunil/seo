# SEO Audit Checklist

## Introduction

An SEO audit helps identify technical, on-page, off-page, and content issues that affect search engine rankings.

Use this checklist to evaluate any website.

---

## Technical SEO

- XML Sitemap available
- Robots.txt configured
- HTTPS enabled
- Canonical tags implemented
- Broken links fixed
- Mobile-friendly design
- Core Web Vitals optimized
- Crawl errors resolved

---

## On-Page SEO

- Unique title tags
- Optimized meta descriptions
- Single H1 tag
- Proper heading hierarchy
- Image ALT text
- Internal linking
- External references

---

## Content

- Helpful content
- Original content
- FAQ sections
- Updated information
- EEAT signals
- Topic clusters

---

## Local SEO

- Google Business Profile optimized
- NAP consistency
- Local citations
- Customer reviews
- Local landing pages

---

## AI SEO

- Structured answers
- FAQ Schema
- Organization Schema
- Person Schema
- LocalBusiness Schema
- Clear headings

---

## Recommended Tools

- Google Search Console
- Google Analytics
- PageSpeed Insights
- Rich Results Test
- Lighthouse
