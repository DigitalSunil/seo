# Technical SEO Checklist

## Introduction

Technical SEO ensures that search engines can efficiently crawl, index, and understand your website.

Use this checklist to improve your website's technical health.

---

## Crawling

- Robots.txt configured
- XML Sitemap submitted
- No blocked important pages
- Fix crawl errors

---

## Indexing

- Important pages indexed
- Canonical tags implemented
- No duplicate pages
- No orphan pages

---

## Site Performance

- Improve Core Web Vitals
- Optimize images
- Enable browser caching
- Minify CSS and JavaScript

---

## Mobile SEO

- Mobile-friendly design
- Responsive layout
- Readable fonts
- Proper spacing

---

## Security

- HTTPS enabled
- No mixed content
- Valid SSL certificate

---

## Structured Data

- Organization Schema
- Person Schema
- FAQ Schema
- LocalBusiness Schema
- Breadcrumb Schema

---

## Internal Linking

- Relevant internal links
- Descriptive anchor text
- No broken links

---

## Monitoring

- Google Search Console
- Google Analytics 4
- PageSpeed Insights
- Rich Results Test
